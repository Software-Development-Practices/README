## Commit Conventions

참고자료: [좋은 git 커밋 메시지를 작성하기 위한 8가지 약속](https://djkeh.github.io/articles/How-to-write-a-git-commit-message-kor/)

- 커밋 메세지는 영문으로 작성한다.
- 제목은 명령조로 작성한다.
- 제목과 본문은 분리해서 작성한다.
- 본문은 무엇을 보다 `어떻게` 와 `왜` 에 집중해서 작성한다.
